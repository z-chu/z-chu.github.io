Git 命令入门到放弃

发表于 2017-03-05 |

创建

  • 创建本地仓库

  • 获取远程仓库

例：git clone https://github.com/you/yourpro.git

远程

  • 添加一个新的 remote 远程仓库

例：git remote add origin https://github.com/you/yourpro.git
origin：相当于该远程仓库的别名

  • 列出所有 remote 的别名

  • 列出所有 remote 的 url

  • 删除一个 renote

    git remote rename [old-name] [new-name]

    git remote set-url origin [url]
// 例如：Git remote set-url origin https://github.com/you/yourpro.git


git remote rm origin
git remote add origin [url]

    git add . // 添加所有文件
    git add file.txt // 添加指定文件

    git rm file.txt // 从版本库中移除，删除文件
    git rm file.txt -cached // 从版本库中移除，不删除原始文件
    git rm -r xxx // 从版本库中删除指定文件夹

    git commit -m "注释"

    // 撤销最近的一个提交.
    git revert HEAD

    // 取消 commit + add
    git reset –mixed

    // 取消 commit
    git reset –soft

    // 取消 commit + add + local working
    git reset –hard

    git push [remote-name] [loca-branch]:[remote-branch]
    例：git push origin master:master

    git status

    git fetch [remote-name]/[branch]

    git pull [remote-name] [branch]

    git merge [remote-name]/[branch]
    //…解决冲突
    git add .
    git commit -m’fix conflict’

    git rebase [remote-name]/[branch]
    //… 解决冲突
    git add .
    git rebase –continue

    // 列出分支
    git branch

    // 创建一个新的分支
    git branch (branch-name)

    // 删除一个分支
    git branch -d (branch-nam)

    // 删除 remote 的分支
    git push (remote-name) :(remote-branch)

    // 切换到一个分支
    git checkout [branch-name]

    // 创建并切换到该分支
    git checkout -b [branch-name]

/.idea // 过滤指定文件夹
/fd/ // 忽略根目录下的 /fd/ 目录的全部内容； .iml // 过滤指定的所有文件
!.gitignore // 不忽略该文件
